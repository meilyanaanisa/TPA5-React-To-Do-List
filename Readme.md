### Development

- create react with vite
```
npm create vite@latest tpa5-project
```

- install react-redux
```
npm install redux react-redux
```

- install toolkit + motion
```
npm install react redux @reduxjs/toolkit framer-motion
```

- Install Depencies
```
npm install
```

- Run Server
```
npm run dev
```

- Build Development
```
npm run build
```
